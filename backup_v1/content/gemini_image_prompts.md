# Gemini 2.0 Flash Image Prompts for Mohawk Medibles Blogs

## Instructions
1. Copy each prompt below into Gemini 2.0 Flash
2. Download the generated image
3. Rename using the filename provided
4. Compress at squoosh.app (WebP, 80% quality)
5. Upload to WordPress with the alt text provided

## WordPress Upload Instructions

1. Go to Media → Add New
2. Upload compressed WebP images
3. Click on each image → Edit
4. Add Alt Text exactly as specified above
5. Add Title (same as alt text)
6. Save

Then insert images into blog posts at appropriate sections.
